# Plan: 목업 전면 리뉴얼 (mockup-renewal)

> 더헤이븐리트리트 웹기획서 v1 기반 목업 재구축

## 1. 배경 및 목적

### 현재 상태
- 기존 "우재네 캠핑" 목업을 스토너리 톤으로 리스킨한 상태
- 메뉴/페이지 구조가 웹기획서와 완전히 다름
- 객실 타입(글램핑돔/카라반/오토캠핑)이 실제와 맞지 않음

### 목표
- 웹기획서 v1의 구조/콘텐츠를 100% 반영한 목업
- 클라이언트 무드보드(벨텐트/티피/사파리텐트) 비주얼 반영
- 스토너리 럭셔리 톤 유지 + 자연 감성 글램핑 융합

## 3. 웹기획서 분석

### 3.1 사이트맵 / 메뉴 구조

```
THE HAVEN RETREAT
├── 메인페이지 (원페이지 스크롤)
│   ├── 히어로 (숲 영상 → 이미지 대체 + "향후 영상으로 교체 예정")
│   ├── 브랜드 소개 ("온전한 Re:creation을 위한 일곱개의 센서리 정원")
│   ├── 일곱개의 정원
│   │   ├── 스카이가든 (데크 사이트 A/B + 파쇄석)
│   │   ├── 스톤가든 (BBQ존 - 바베큐장)
│   │   ├── 윈드가든 (이벤트/테라스)
│   │   ├── 케이브가든 (레스토랑)
│   │   ├── 워터가든 (수영장/스파)
│   │   └── 포레스트가든 (산책로/트리하우스)
│   ├── MAP (배치도)
│   └── 이벤트 카드
│
├── 프롤로그 (Prologue) — 브랜드 스토리
│
├── 캠핑장
│   ├── 사이트 A (2인/최대4인, 데크 6x8, 공용샤워실, 수영장)
│   └── 사이트 B (4인/최대6인, 데크 12x8, 공용샤워실, 수영장)
│
├── 바베큐존
│   ├── 바베큐 소개 (직원이 직접 육성한 고기 / 셀프바)
│   ├── 옵션 A-1 (₩200,000 풀서비스 - 직원이 구워줌)
│   ├── 옵션 A-2 (₩150,000 셀프)
│   ├── 옵션 B-1 (₩140,000 풀서비스)
│   └── 옵션 B-2 (₩100,000 셀프)
│
├── SHOP — (추후 확장)
├── 대관 — 대관 안내 이미지
├── 문의 — (연락처/문의폼)
│
├── 이벤트 — 진행/예정 이벤트 카드 목록
│
├── 예약하기
│   ├── 캠핑장 예약 및 이용안내
│   └── 바베큐장 예약 및 이용안내
│
└── 푸터 (주소, TEL, Instagram, Copyright)
```

### 3.2 회원 체계

| 구분 | 로그인 전 | 일반회원 | 파트너회원 |
|------|----------|---------|---------|
| 공지사항/이벤트/홈콕정보/FAQ | O | O | 파트너용 별도 |
| 개인정보/결제수단/적립금/쿠폰 | X (alert) | O | O |
| 1:1 문의 | X | O (일반) | O (파트너) |
| 회원탈퇴 | - | O | X (메뉴없음) |
| 마이페이지 프로필란 | 로그인창 | 회원정보 | 파트너정보 |

### 3.3 주요 기능 요구사항

1. **메인페이지**: 스크롤 기반 원페이지, 클릭 시 해당 섹션으로 부드러운 이동
2. **캠핑장 페이지**: 이미지 자동 슬라이드(4초), 좌우 화살표 수동 이동
3. **바베큐 페이지**: 이미지 자동 슬라이드, 관리자에서 이미지/워딩 수정 가능
4. **이벤트 페이지**: 클릭 시 카테고리별 필터, 상세 페이지 이동
5. **예약**: 캠핑장/바베큐장 각각 별도 예약 flow
6. **관리자**: 이미지 및 워딩 수정 가능하도록

### 3.4 이용안내 상세

**캠핑장:**
- 체크인 15:00 / 체크아웃 11:00
- 22:00 이후 체크인 시 사전 연락
- 예약 후 12시간 내 결제 (당일/하루 전은 6시간)
- 미결제 시 자동 취소
- 최대인원 초과 시 이용/환불 불가
- 인원수 맞춤 조식 제공
- 공용 샤워실 + 수영장(하절기)
- 금연, 애완동물 출입 금지

**바베큐장:**
- 런치 11:00~14:00 / 디너 17:00~20:00
- A-1/B-1 이용객은 점검 면제
- 동일 결제/취소 정책

## 4. 디자인 방향

### 4.1 비주얼 스타일
- **레이아웃**: 스토너리(stonery.kr) 참조 — 풀스크린 섹션, 미니멀, 럭셔리
- **색상**: 브라운(#4c4543) + 베이지골드(#e6cfac) + 화이트 + 자연 그린
- **폰트**: Cormorant Garamond (영문 세리프) + Noto Sans KR
- **이미지 톤**: 클라이언트 무드보드 — 벨텐트/티피/사파리텐트 on 우드 데크, 숲속 자연 감성

### 4.2 디자인 도구
- **Google Stitch MCP**: 주요 섹션별 UI/UX 디자인 생성에 활용
- **Gemini API**: 실제 현장 사진 기반 마케팅 이미지 생성 (이미 진행됨)
