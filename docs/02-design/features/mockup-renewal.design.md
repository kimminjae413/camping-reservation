## 1. 디자인 시스템

### 1.1 색상 팔레트 (스토너리 기반 + 자연 감성)

```
Primary:
  --haven-brown:    #4c4543   (메인 브랜드 컬러)
  --haven-beige:    #e6cfac   (악센트/골드)
  --haven-cream:    #faf6f1   (배경)
  --haven-dark:     #2a2523   (푸터/다크)

Nature:
  --haven-forest:   #3d5a3d   (자연 그린 - 포인트용)
  --haven-moss:     #6b7c5e   (모스 그린)
  --haven-stone:    #8a8580   (스톤 그레이)

Neutral:
  --haven-white:    #ffffff
  --haven-gray-100: #f5f4f3
  --haven-gray-300: #d4d0cc
  --haven-gray-600: #7a7572
  --haven-gray-900: #2d2926
```

### 1.2 타이포그래피

```
영문 헤딩:  Cormorant Garamond (300, 400, 600) — 세리프, 럭셔리
한글 본문:  Noto Sans KR (300, 400, 500, 700) — 가독성
영문 본문:  Noto Sans KR과 동일

크기 스케일:
  hero-title:    72px / 4.5rem (모바일: 40px)
  section-title: 40px / 2.5rem (모바일: 28px)
  sub-title:     24px / 1.5rem
  body:          16px / 1rem
  caption:       14px / 0.875rem
  label:         12px / 0.75rem
```

### 1.3 간격 & 레이아웃

```
max-width: 1280px (컨텐츠 영역)
섹션 패딩: py-24 (96px) / 모바일 py-16
카드 간격: gap-8 (32px)
컨텐츠 px: px-6 (24px) / 모바일 px-4
```

### 1.4 컴포넌트 스타일

```
버튼-primary:   bg-[#4c4543] text-white rounded-none tracking-widest uppercase
버튼-secondary: border border-[#4c4543] text-[#4c4543] rounded-none
버튼-accent:    bg-[#e6cfac] text-[#4c4543] rounded-none
카드:           rounded-none shadow-none border border-gray-200 (미니멀)
이미지:         rounded-none (각진 스타일, 스토너리 톤)

```

## 5. 인터랙션 & 애니메이션

### 5.1 스크롤 효과
- 섹션 진입 시: fade-in + translateY(20px → 0)
- 이미지: parallax 효과 (선택적)
- 네비게이션: 현재 섹션 하이라이트

### 5.2 이미지 슬라이드
- 캠핑장/바베큐존: 4초 자동 슬라이드
- 좌우 화살표 + 하단 인디케이터 dots
- 스와이프(모바일)

### 5.3 트랜지션
- 모든 트랜지션: cubic-bezier(.77,0,.18,1) (스토너리 이징)
- duration: 0.3s ~ 0.7s
